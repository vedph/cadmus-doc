# Developing Backend

When creating the backend for a Cadmus project, typically you will create a backend Visual Studio solution to host your optional project-specific models (parts and fragments with their seeders and services) and your API.

The API project will just be a wrapper for imported logic except for the configuration, which will be specific to your project.
