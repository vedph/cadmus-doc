# Logging in Cadmus API

Logging in Cadmus API has been streamlined and the MongoDB sink has been removed. Should you need to log to a database, enable the already existing PostgreSQL logging option in `appsettings.json`.

So, ensure that:

- you remove all references to package `Serilog.Sinks.MongoDB`, as they are no longer required.
- in `appsetting.json`, remove from `Serilog.Using` array the entry for MongoDB if present.
