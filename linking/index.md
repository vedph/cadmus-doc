# Linking Data

Linking data is an essential task to create networks of objects. In Cadmus, links refer to these main areas:

- _linking the editor to closed sets of data_, e.g. a dropdown control in its UI to a list of languages for a language property. This is done using **thesauri**.
- _linking the editor to independent taxonomies_. The purpose of this link is the same as that of thesauri: what changes is scale and editing strategy.
- _linking a part to an item or any of its parts or to a specific property in any of its parts_. This is done via **internal links**.
- _linking a part or a property of it to an external resource_. This is done via **external links**.
